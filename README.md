# se-skill

一组软件工程相关的 skill。

推荐使用 npx skills 安装：
```shell
npx skills add XLCYun/se-skills
```

## bad-smells-skill

`bad-smells-skill` 主要从重构坏味道的视角做审查，关注设计质量与可维护性问题。它适合用于检查仓库、子系统或变更集，并将发现整理为结构化审查报告和按优先级排序的后续整改项。

## anti-pattern-skill

`anti-pattern-skill` 用来识别更广义的软件与交付反模式，覆盖代码、测试、架构和实现方式等层面。

这个 Skill 目前还处于调整状态。
